## MAC address changer
*Program, ktorý mení MAC adresu zvoleného interface, na zvolenú novú MAC adresu.*  
#### Syntax spustenia 
`$ python3 mac_changer.py [-i / --interface] interface_name [-m / --mac] new_mac_address`
